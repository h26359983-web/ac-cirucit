15.1 مقدمة — الممانعة في دوائر AC

قانون أوم للتيار المتناوب

V = I × Z (الجهد = التيار × الممانعة) I = V / Z Z = V / I Z تُقاس بالأوم (Ω) وهي عدد مركّب

أنواع الممانعة الثلاثة

Z_R = R ∠0° (مقاومة: V وI في نفس الطور) Z_L = X_L ∠+90° (ملف: V يتقدم I بـ 90°) Z_C = X_C ∠-90° (مكثّف: I يتقدم V بـ 90°)

المقاومة النقية (R فقط)

v = Vm·sin(ωt) → V = V∠0° I = V∠0° / R∠0° = (V/R)∠0° V وI متزامنان — زاوية الطور θ = 0°
📌 الفرق بين AC وDC: في AC الجهد والتيار يتغيران بشكل جيبي مع الزمن، لذا نستخدم المتجهات الطورية (Phasors)

التحويل بين القيم الفعالة والذروة

V_rms = V_m / √2 = 0.707 × V_m I_rms = I_m / √2 = 0.707 × I_m V_m = √2 × V_rms = 1.414 × V_rms
15.2 الممانعة الحثية — Inductive Reactance XL

معادلة X_L

X_L = ω·L = 2π·f·L (أوم) Z_L = X_L ∠+90° = jX_L حيث: f = التردد (Hz), L = الحث الذاتي (Henry)

علاقة X_L بالتردد

X_L ∝ f (كلما زاد التردد زاد X_L) عند f=0 (DC): X_L = 0 → الملف = سلك عادي عند f=∞: X_L = ∞ → الملف = دائرة مفتوحة

مثال حسابي

X_L = 4Ω, v = 24sin(ωt)V V_rms = 24/√2 = 16.97V ∠0° I = V/Z_L = 16.97∠0° / 4∠90° = 4.24A ∠-90° i = 4.24×√2 × sin(ωt-90°) = 6A·sin(ωt-90°) الملاحظة: التيار متأخر عن الجهد بـ 90°
💡 الملف يخزن الطاقة في المجال المغناطيسي — لا يستهلكها. القدرة الحقيقية للملف = صفر
15.3 الممانعة السعوية — Capacitive Reactance XC

معادلة X_C

X_C = 1/(ω·C) = 1/(2π·f·C) (أوم) Z_C = X_C ∠-90° = -jX_C حيث: C = السعة الكهربائية (Farad)

علاقة X_C بالتردد

X_C ∝ 1/f (كلما زاد التردد قلّ X_C) عند f=0 (DC): X_C = ∞ → المكثف = دائرة مفتوحة عند f=∞: X_C = 0 → المكثف = سلك عادي

مثال حسابي

X_C = 2Ω, i = 5sin(ωt+30°)A I_rms = 5/√2 = 3.54A ∠+30° V = I × Z_C = 3.54∠30° × 2∠-90° = 7.07V ∠-60° v = 10·sin(ωt-60°)V الملاحظة: التيار يتقدم الجهد بـ 90°
💡 المكثف يخزن الطاقة في المجال الكهربائي — لا يستهلكها. القدرة الحقيقية للمكثف = صفر
15.4 دوائر R-L-C التسلسلية

الممانعة الكلية Z_T

Z_T = R + j(X_L - X_C) (شكل مستطيل) |Z_T| = √(R² + X_net²) (مقدار الممانعة) θ = arctan(X_net / R) (زاوية الطور) X_net = X_L - X_C (الممانعة الصافية)

مثال: دائرة R-L (R=3Ω, X_L=4Ω, E=100V∠0°)

Z_T = 3+j4 = 5∠53.13° Ω I = E/Z_T = 100∠0° / 5∠53.13° = 20A ∠-53.13° V_R = I×R = 20∠-53.13° × 3∠0° = 60V ∠-53.13° V_L = I×X_L = 20∠-53.13° × 4∠90° = 80V ∠+36.87° تحقق: V_R² + V_L² = 60²+80² = 10000 = 100² ✓

الطاقة والقدرة

P = I²·R = E·I·cos(θ) [Watt] — قدرة حقيقية Q = I²·X_net [VAR] — قدرة خيالية S = E·I = I²·Z_T [VA] — قدرة ظاهرية PF = cos(θ) = R/|Z_T| — معامل القدرة S² = P² + Q² — مثلث القدرة
📌 من Chapter 15: مثلث القدرة مشابه لمثلث الممانعة: P=IR, Q=IX_net, S=IZ_T
15.5 قاعدة منقسم الجهد (VDR)

قانون منقسم الجهد للتيار المتناوب

V_x = (Z_x / Z_T) × E حيث: V_x = الجهد على العنصر x Z_x = ممانعة العنصر x (بشكل متجه) Z_T = الممانعة الكلية (بشكل متجه) E = جهد المصدر

مثال 1: دائرة R-C (R=3Ω, X_C=4Ω, E=100V∠0°)

Z_T = 3-j4 = 5∠-53.13° V_R = (3∠0° / 5∠-53.13°) × 100 = 60V ∠+53.13° V_C = (4∠-90° / 5∠-53.13°) × 100 = 80V ∠-36.87° تحقق: V_R² + V_C² = 3600+6400 = 10000 = E² ✓

مثال 2: دائرة R-L-C (R=6Ω, X_L=9Ω, X_C=17Ω, E=50V∠30°)

X_net = 9-17 = -8Ω (سعوية) Z_T = 6-j8 = 10∠-53.13° I = 50∠30° / 10∠-53.13° = 5A ∠83.13° V_R = 5×6∠83.13° = 30V ∠83.13° V_L = 5×9∠(83.13°+90°) = 45V ∠173.13° V_C = 5×17∠(83.13°-90°) = 85V ∠-6.87°
15.6 الرنين والاستجابة الترددية

شرط الرنين التسلسلي

X_L = X_C (شرط الرنين) ω_r·L = 1/(ω_r·C) ω_r = 1/√(L·C) (rad/s) f_r = 1/(2π·√(L·C)) (Hz)

خصائص دائرة الرنين التسلسلية

عند الرنين: • Z_T = R (أدنى ممانعة) • I_max = E/R (أقصى تيار) • PF = 1 (cos 0° = 1) • θ = 0° (V وI في نفس الطور) • V_L = V_C = Q×E (قد تتجاوز E!)

عامل الجودة Q

Q = X_L/R = X_C/R = ω_r·L/R = 1/(ω_r·R·C) Q كبير → نطاق ترددي ضيق → تضخيم جهد أكبر عند الرنين: V_L = V_C = Q × E

الاستجابة الترددية R-C

f_c = 1/(2π·R·C) (تردد الانقطاع) عند f = f_c: |H| = 1/√2 = 0.707 = -3dB
⚠️ تحذير: عند الرنين، جهد الملف والمكثف قد يتجاوز جهد المصدر بعامل Q! قد يُتلف المكونات.
15.7 الدوائر التوازية للتيار المتناوب

تحليل الدوائر التوازية

Z_T = (Z₁×Z₂)/(Z₁+Z₂) (مكوّنان متوازيان) أو: 1/Z_T = 1/Z₁ + 1/Z₂ + ... قاعدة منقسم التيار: I_x = (Z_T / Z_x) × I_T

المُقبولية Y (Admittance)

Y = 1/Z = G + jB (سيمنز) G = Conductance = R/|Z|² (موصلية) B = Susceptance = -X/|Z|² (إقبالية) B_L = -1/X_L (سالبة) B_C = +1/X_C (موجبة) I_T = E × Y_T

الرنين التوازي

f_r = 1/(2π·√(LC)) (تقريباً) عند الرنين التوازي: • Z_T = L/(R·C) = أقصى ممانعة • I_T = أدنى تيار (عكس التسلسلي!) • التيارات الداخلية كبيرة جداً
📌 الرنين التسلسلي: Z_min, I_max | الرنين التوازي: Z_max, I_min — عكس تام!
